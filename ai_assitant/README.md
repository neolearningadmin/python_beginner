# Simple OpenAI Assistant (Python)

This is a minimal example of a local AI assistant using the OpenAI API.

Files:
- `assistant.py` - small wrapper around the OpenAI ChatCompletion API
- `main.py` - simple CLI loop to talk to the assistant
- `requirements.txt` - Python dependencies
- `key.py` - Place key here. Do not share this key, or upload to Github.

Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

Run

```bash
python main.py
```

Type messages and press Enter. Use `exit` or `Ctrl-C` to quit.
